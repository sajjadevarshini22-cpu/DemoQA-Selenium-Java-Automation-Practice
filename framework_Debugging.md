# Framework Debugging and Common Errors

## Errors Practiced
- NoSuchElementException
- ElementClickInterceptedException
- TimeoutException
- StaleElementReferenceException
- NullPointerException
- SessionNotCreatedException
- Maven dependency errors
- TestNG No tests found
- Screenshot path errors
- Extent Report errors

## Debugging Checklist
1. Read error message.
2. Check line number.
3. Check locator.
4. Check wait.
5. Check test data.
6. Check config.properties.
7. Check driver initialization.
8. Check BaseTest inheritance.
9. Check testng.xml.
10. Check Maven dependencies.
11. Check reports and screenshots folders.
